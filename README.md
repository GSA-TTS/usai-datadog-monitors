# usai-datadog-monitors

Datadog monitor configurations for the USAi platform, tracked as code.

## Structure

```
monitors/
  aigov/          # fcs-mcaas-aigov.ddog-gov.com
  gsai/           # fcs-mcaas-gsai.ddog-gov.com
```

Each monitor is stored as a JSON file matching the Datadog API format.
