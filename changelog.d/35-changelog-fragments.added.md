CHANGELOG fragment-file workflow (`changelog.d/` + `scripts/assemble-changelog.sh`) to end the `### Added` three-way-merge tax (#35). Each PR now drops a standalone `changelog.d/<id>-<slug>.<type>.md` fragment instead of appending to the shared list head in `CHANGELOG.md`, so sibling PRs never collide (four consecutive monitor PRs — #29/#30/#31/#32 — each paid that merge tax). At release, `scripts/assemble-changelog.sh` folds the fragments into `[Unreleased]`, grouped by Keep a Changelog type, and deletes them; `--check` lists pending fragments without touching anything. Toolchain-free (POSIX sh + awk, no Python added to a Terraform repo).
